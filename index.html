Yeezy Taught Me Text Generation:

Train a LSTM (Long Short Term Memory) model to generate text.

DESCRIPTION

This example allows you to train a model to generate text in the style of some existing source text. The model is designed to predict the next character in a text given some preceding string of characters. Doing this repetedly builds up a text, character by character.

As Kanye West said "Lack of visual empathy, equates the meaning of L-O-V-E."
